# Grocery List Project

## Description
The Grocery List Project is a simple application aimed at helping users manage their grocery shopping needs efficiently. The application provides a user-friendly interface where users can create, edit, and manage their grocery lists. It allows users to add items to their lists, mark items as purchased, and remove items they no longer need.

## Features
- **User Authentication**: Users can sign up and log in securely to access their grocery lists.
- **Create and Manage Lists**: Users can create multiple grocery lists for different purposes (e.g., weekly shopping, special occasions).
- **Add and Remove Items**: Users can easily add new items to their lists and remove items they no longer need.
- **Mark Items as Purchased**: Users can mark items as purchased once they have bought them.
- **Search and Filter**: Users can search for specific items within their lists and filter items by category or other criteria.
- **Customization**: Users can customize their lists by organizing items, setting priorities, and adding notes.
- **Share Lists**: Users can share their lists with family members or friends for collaborative shopping.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript, React
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JSON Web Tokens (JWT)
- **Version Control**: Git, GitHub
